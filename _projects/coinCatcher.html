Introduction

Are you ready to challenge the monsters at a coin gathering game? In this a game you are playing as a robot whose goal is to gather as many raining coins as possible.

The game was created as a final project of the MOOC Programming course. Only given images (hirvio.png, kolikko.png, ovi.png and robo.png) were allowed to use and everything else should be created using Python.
