Introduction

Koiratarha 2.0 (DogPark 2.0) is a web application that uses MyHelsinki Open API to show various public dog parks and other dog related areas on a map.

A user can register to the site to save their favourite locations and set reminders for visits to those locations. The app is a convenient place for all dog owners to find nearby and fun places to visit with their dogs.
